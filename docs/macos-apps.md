## üíª Development

### Warp - AI Terminal
```
What: Modern terminal with AI assistance
When: Daily terminal use

Features:
- AI command suggestions (type # to ask)
- Block-based output (select, copy easily)
- Command search (Ctrl+R)
- Split panes
- Team sharing

Pro tips:
- # how do I find large files
- Click output blocks to select
- Ctrl+Shift+D for vertical split
```

### Antigravity - AI Coding
```
What: AI coding assistant
When: Complex coding tasks

Features:
- Code generation
- Debugging help
- Refactoring suggestions
- Multi-file edits
```

### Visual Studio Code
```
What: Code editor with extensions
When: Editing code, configs

Essential extensions:
- GitLens
- Docker
- Kubernetes
- Terraform
- Remote SSH
- Error Lens

Keybindings:
- Cmd+P = Quick open file
- Cmd+Shift+P = Command palette
- Cmd+B = Toggle sidebar
- Cmd+` = Terminal
```

### Sublime Text
```
What: Fast text editor
When: Quick edits, large files

Why use it:
- Opens instantly
- Handles huge files
- Multiple cursors
- Distraction-free mode

Keybindings:
- Cmd+D = Select next occurrence
- Cmd+Shift+L = Split selection to lines
- Cmd+P = Goto anything
```

### OrbStack
```
What: Docker & Linux on Mac
When: Container development

Benefits over Docker Desktop:
- 2x faster startup
- 50% less memory
- Native Linux VMs
- Rosetta for x86

Usage:
- Same docker commands work
- docker-compose works
- Kubernetes included
```

### Lens
```
What: Kubernetes IDE
When: Managing K8s clusters

Features:

- Log streaming
- Shell access
- Prometheus metrics

When to use vs k9s:
```
